## AI Image Tagging

This fork adds on-device AI image tagging using Google's Gemma model via LiteRT. Tags are generated entirely offline — no data leaves your device.

**Model required:** `gemma-4-E2B-it.litertlm`  
**Download from:** https://huggingface.co/litert-community/gemma-4-E2B-it-litert-lm?library=litert-lm

**Installation:**
1. Transfer the model file to your phone (e.g. via USB to Downloads)
2. Open the app, tap any image, and go to **Tags → Import AI Model**
3. Select the model file — the app will copy it into place

Once the model is in place, open the app and use **Tags → Generate & Save AI Tags** on any image, or **Tags → Bulk Tag All Images** to tag your entire library in the background.
